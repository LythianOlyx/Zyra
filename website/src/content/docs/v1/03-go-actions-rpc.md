# Go Actions RPC Protocol

The **Go Action RPC Protocol** is Zyra's core data-fetching mechanism. It completely eliminates manual API endpoints, REST controllers, manual TypeScript interfaces, and boilerplate serialization code.

## Defining a Go Action

Write a standard Go function in your `app/actions` package and add the `// +zyraaction` comment directive:

```go
package actions

import (
    "context"
    "github.com/LythianOlyx/Zyra/pkg/zyra"
)

type CreateUserInput struct {
    Name     string `json:"name" validate:"required,min=2"`
    Email    string `json:"email" validate:"required,email"`
    Password string `json:"password" validate:"required,min=8"`
}

type UserDTO struct {
    ID        string `json:"id"`
    Name      string `json:"name"`
    Email     string `json:"email"`
    CreatedAt string `json:"createdAt"`
}

// +zyraaction
func CreateUser(ctx context.Context, input CreateUserInput) (*UserDTO, error) {
    // Auth guard example
    session := zyra.Auth.MustUser(ctx)

    // Save to database
    user, err := db.Users.Create(ctx, input.Name, input.Email, input.Password)
    if err != nil {
        return nil, zyra.NewError(400, "User creation failed: "+err.Error())
    }

    return &UserDTO{
        ID:        user.ID,
        Name:      user.Name,
        Email:     user.Email,
        CreatedAt: user.CreatedAt.Format("2006-01-02"),
    }, nil
}
```

## Automatic Code Generation

When `zyra dev` or `zyra build` runs, Zyra inspects all `// +zyraaction` directives and generates fully typed TypeScript definitions and React hooks in `.generated/`:

```typescript
// Auto-generated by Zyra CLI — DO NOT EDIT DIRECTLY

export interface CreateUserInput {
  name: string;
  email: string;
  password: string;
}

export interface UserDTO {
  id: string;
  name: string;
  email: string;
  createdAt: string;
}

export const CreateUser = createZyraAction<CreateUserInput, UserDTO>('CreateUser');
```

## Consuming Go Actions in React

Import the generated hook directly into your React component with 100% end-to-end type safety and autocomplete:

```tsx
import React, { useState } from 'react';
import { useZyraAction } from '@/runtime/client';
import { CreateUser } from '@/.generated/actions';

export function RegisterForm() {
  const [name, setName] = useState('');
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');

  const { execute, data, loading, error } = useZyraAction(CreateUser);

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    const user = await execute({ name, email, password });
    if (user) {
      alert(`User created: ${user.name} (${user.id})`);
    }
  };

  return (
    <form onSubmit={handleSubmit} className="space-y-4">
      <input 
        value={name} 
        onChange={(e) => setName(e.target.value)} 
        placeholder="Full Name" 
      />
      <input 
        value={email} 
        onChange={(e) => setEmail(e.target.value)} 
        placeholder="Email Address" 
      />
      <input 
        type="password" 
        value={password} 
        onChange={(e) => setPassword(e.target.value)} 
        placeholder="Password" 
      />
      <button type="submit" disabled={loading}>
        {loading ? 'Registering...' : 'Register Account'}
      </button>
      {error && <p className="text-red-500">{error.message}</p>}
    </form>
  );
}
```

## Realtime Streaming Actions (`// +zyrastream`)

For live dashboards, chat rooms, or real-time progress indicators, annotate functions with `// +zyrastream`:

```go
// +zyrastream
func LiveSystemMetrics(ctx context.Context, stream *zyra.Stream[MetricsData]) error {
    ticker := time.NewTicker(1 * time.Second)
    defer ticker.Stop()

    for {
        select {
        case <-ctx.Done():
            return nil
        case <-ticker.C:
            metrics := getSystemMetrics()
            stream.Send(metrics)
        }
    }
}
```

In React:

```tsx
const { data, isConnected } = useZyraStream(LiveSystemMetrics);
```
